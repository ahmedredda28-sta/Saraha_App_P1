# Saraha Structure

## Overview
Saraha Structure is a project designed to manage user authentication and database interactions. It provides a structured approach to handling user data and authentication processes.

## Project Structure
```
Saraha_Structure
├── src
│   ├── DB
│   │   ├── index.js
│   │   ├── database.repository.js
│   │   ├── connection.db.js
│   │   └── models
│   │       └── index.js
│   ├── user.model.js
│   ├── auth.controller.js
│   ├── auth.service.js
├── package.json
└── README.md
```

## Features
- User registration and login functionality.
- Database connection and repository for data management.
- Modular structure for easy maintenance and scalability.

## Installation
To install the necessary dependencies, run:
```
npm install
```

## Usage
To start the application, use:
```
npm start
```
